랜딩페이지를 만들어 볼껀데 구성은 헤더, 메인배너, 검색창, 4*4 카드섹션, 푸터로 구성되게 할거야
전체적인 사이즈 : PC 1920 x 1080, 모바일 웹사이트/앱 디자인: 스마트폰: 360 x 640px, 375 x 667px (iPhone), 414 x 896px (iPhone Plus/Max) 태블릿: 768 x 1024px (iPad), 1024 x 1366px (iPad Pro)에 최적화 되게 구성해줘
헤더 :  좌측에 "덕분이의 골프맛집"이라는 로고 텍스트를 넣어주고 로고텍스트를 누르면 홈으로 이동하게 설정해줘, 메인배너의 좌측 끝에 위치하게 만들어줘야해. 헤더의 너비는 카드섹션의 좌우측 끝선에 맞줘서 조절해줘
메인배너 :  사이즈는 너비 1080*400사이즈로 해주고 URL을 넣어서 이미지가 삽입될 수 있게 해주고 이미지를 누르면 https://food.dokbun2.com/로 연결 될 수 있도록 해줘
검색창 : 검색창은 카드섹션의 내용을 검색할 수 있게 해주고 검색창의 결과 값은 카드섹션 단일로 보여줄 수 있도록 해줘, 검색창의 너비도 카드섹션의 좌우측 끝선에 맞도록 넓혀줘
카드섹션 : 4*4로 카드섹션을 구성해주고 카드섹션을 누르면 url을 통해 다른 사이트로 접속할 수 있게 해줘, 카드섹션에는 골프장 이름만 들어갈거고 다른 건 들어가는게 없어
푸터 : 간단한 소개글을 넣어서 골프치기전 골프장 맛집을 찾아야하는 이유를 간략하게 소개할거야
글꼴 : 글꼴은 fonts폴더에 있는 Paperlogy폰트로 적용할껀데 총 9가지이고 TTF형식으로 되어 있어 적용해줘

각 헤더, 메인배너, 검색창, 카드섹션, 푸터사이의 간격은 아래와 같이 해줘
헤더와 메인 배너 사이: 30px
메인 배너와 검색창 사이: 40px
검색창 내부 패딩: 10px
검색 버튼과 입력 필드 사이: 10px

디자인 : 골프장과 관련된 디자인으로 깔끔하고 시원한 느낌으로 해주는 색상으로 해줘
메인배너의 가로너비를 검색창의 가로너비와 같게 해줘
